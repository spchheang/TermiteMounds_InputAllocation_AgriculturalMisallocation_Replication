### Chrey Bak Catchment Boundary Data

This project uses a boundary file for the **Chrey Bak Catchment**, located in **Kampong Chhnang Province, Cambodia**. The boundary is provided as a KML file:

```text
CB_District.kml
```

The file contains the district-level boundary used to identify and map the Chrey Bak Catchment study area. It is used to create maps showing the location and extent of the Chrey Bak Catchment in Kampong Chhnang Province.

#### Purpose of the Dataset

The `CB_District.kml` file is used to define the geographic boundary of the Chrey Bak Catchment study area. This boundary is used for mapping, spatial visualization, and identifying the administrative districts included in the catchment area.

The file can be opened and processed in GIS software such as QGIS, ArcGIS, R, or Python.
