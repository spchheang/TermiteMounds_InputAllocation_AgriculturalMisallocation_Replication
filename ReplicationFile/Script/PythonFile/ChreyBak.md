### Chrey Bak Catchment Boundary Data

This project uses boundary and village reference files for the **Chrey Bak Catchment**, located in **Kampong Chhnang Province, Cambodia**. These files are used to define the study area, identify villages included in the analysis, and create maps for the Chrey Bak Catchment.

The main boundary file is provided as a KML file:

```text
CB_District.kml
```

The `CB_District.kml` file contains the district-level boundary used to identify and map the Chrey Bak Catchment study area. It is used to create maps showing the location and extent of the Chrey Bak Catchment in Kampong Chhnang Province.

This project also uses the following village reference file:

```text
TM_Village.csv
```

The `TM_Village.csv` file is created from the data used in the paper. It contains the village names in the Chrey Bak Catchment study area and is used as a reference file to identify, match, and map villages included in the analysis. This file can also be used to check the number of villages in the study area.

#### Purpose of the Dataset

The `CB_District.kml` file is used to define the geographic boundary of the Chrey Bak Catchment study area. This boundary is used for mapping, spatial visualization, and identifying the administrative districts included in the catchment area.

The `TM_Village.csv` file is used to reference the village names included in the study area. It helps match the villages from the paper data with spatial village boundaries and supports mapping or checking the number of villages in the Chrey Bak Catchment.

These files can be opened and processed in GIS software or programming tools such as QGIS, ArcGIS, R, or Python.
